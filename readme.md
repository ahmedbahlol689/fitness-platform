# FitCoach AR

تطبيق Web + PWA عربي كنسخة أولى لمدرب جيم وتغذية للمبتدئين والمتوسطين.

## الموجود في النسخة الأولى

- بروفايل المتدرب: العمر، الجنس، الطول، الوزن، النشاط، الهدف، نظام التدريب.
- حساب السعرات والماكروز اليومية.
- خطط 12 أسبوع: PPL و Upper/Lower و Full Body و Bro Split.
- تمرين اليوم مع تسجيل الوزن والعدات والمجموعات والملاحظات.
- مكتبة تمارين عربية بشرح الأداة والعضلة والأخطاء الشائعة.
- لوحة تقدم للالتزام وحجم التمرين وأعلى وزن.
- حفظ تلقائي في المتصفح عبر localStorage.
- PWA قابلة للتثبيت على الموبايل والكمبيوتر.

## التشغيل

```bash
npm install
npm run dev
```

## بناء نسخة الإنتاج

```bash
npm run build
npm run preview
```

## الخطوة القادمة

ربط Supabase للتسجيل الحقيقي ومزامنة البيانات بين الأجهزة، ثم تجهيز نسخة المتاجر عبر Capacitor أو React Native.
